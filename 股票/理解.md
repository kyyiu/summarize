# 股票
## 最高价最低价    
最高价就是一天中买方力量达到的极致时的价格。A股对于涨幅有10%的限制，最多不过涨停。涨停意味着任何卖出都会被瞬间消化，你想买也很难买到。
最低价反之。最多不过跌停，这时，不是你想卖就能卖掉的。

## 收盘价    
收盘价意味着一天下来多空双方博弈终于有了结果：如果收盘价高于开盘价，多方获胜；反之则空方占优。

## 阳线    
K线最上方的一条细线称为上影线，中间的一条粗线为实体。下面的一条细线为下影线。
当收盘价高于开盘价，也就是股价走势呈上升趋势时，我们称这种情况下的K线为阳线，中部的实体用红色或空白表示。
上影线长度表示最高价和收盘价之间的价差，实体的长短代表收盘价与开盘价之间的价差，下影线的长度则代表开盘价和最低价之间的差距。

## 阴线    
当收盘价低于开盘价，也就是股价走势呈下降趋势时，我们称这种情况下的K线为阴线，中部的实体为绿色。
上影线的长度表示最高价和开盘价之间的价差，实体的长短代表开盘价比收盘价高出的幅度，下影线的长度则由收盘价和最低价之间的价差大小所决定。

## 涨跌
阳线和阴线与个股的涨跌有所不同。一般我们讲的涨跌是指当日收盘价与上个交易日收盘价之问的比较。当K线为阳线时，并不意味着股价比前一天涨了，只是表示当天收盘价高于当天开盘价。

## 意义
阳线表示买盘较强，卖盘较弱，这时由于股票供不应求，会导致股价的上扬；(卖的少想买到就提价让别人想出货)
阴线表示卖盘较强，买盘较弱。此时，由于股票的持有者急于抛出股票，致使股价下挫。(买的少想卖掉就降价让别人买)
上影线越长，表示上面的卖压越强，即意味着股价上升时，会遇到较大的抛售压力；
下影线越长，表示下面的支撑力越强，意味着股价下跌时，会有较多的投资者利用这一机会买进股票。
一根K线图形由蜡烛实体和影线的上下线段组成，影线最上端是最高价，最下端为最低价。红色的是阳线，实体上端的收盘价高于下端的开盘价；绿色阴线则是实体上端的开盘价高于下端的收盘价。

90%筹码集中度
变小主力资金变多散户变少
变多主力资金变少散户变多
出货第一天的集中度乘2就是逃顶的大概值
集中度前面在变小突然变大即为出货



# 期权价格影响因素
## 标的水平
```
call中标的越高于行权价，权利金越高
put中标的越小于行权价，权利金越高
```
## 到期时间
```
到期时间越久，达到可行权的机会越大，价值越高，反之
```
## 标的波动
```
波动越大，达成可行权的机会越大价值越高，反之
```

# 希腊字母
## Delta
```
意义：衡量期权价格对标的资产价格变化的敏感度
范围：看涨期权0到1，看跌期权-1到0
示例：
看涨期权Delta=0.6：标的股价上涨1元，期权价格上涨0.6元
看跌期权Delta=-0.4：标的股价上涨1元，期权价格下跌0.4元
看涨期权Delta=0.6：标的股价下跌1元，期权价格下跌0.6元
看跌期权Delta=-0.4：标的股价下跌1元，期权价格上涨0.4元
```
## Gamma
```
意义：衡量Delta对标的资产价格变化的敏感度(二阶导数)
范围：对所有期权都为正
示例：
看涨期权Gamma=0.05：当股价变动1元，Delta变化0.05
看跌期权Gamma=0.03：当股价变动1元，Delta变化0.03
```

## Theta 
```
意义：衡量期权价格随时间衰减的速度
范围：通常为负(时间损耗)
示例：
看涨期权Theta=-0.05：每天价值减少0.05元
看跌期权Theta=-0.03：每天价值减少0.03元
```

## Vega 
```
意义：衡量期权价格对波动率变化的敏感度
范围：对所有期权都为正
示例：
看涨期权Vega=0.12：波动率增加1%，期权价格上涨0.12元
看跌期权Vega=0.10：波动率增加1%，期权价格上涨0.10元
看涨期权Vega=0.12：波动率减少1%，期权价格下跌0.12元
看跌期权Vega=0.10：波动率减少1%，期权价格下跌0.10元
```

## Rho
```
意义：衡量期权价格对利率变化的敏感度
范围：看涨为正，看跌为负
计算公式: Rho*（利率变化，上升为正，下降为负）
示例：
看涨期权Rho=0.08：利率上升1%，期权价格上涨0.08元
看跌期权Rho=-0.05：利率上升1%，期权价格下跌0.05元
看涨期权Rho=0.08：利率下跌1%，期权价格下跌0.08元
看跌期权Rho=-0.05：利率下跌1%，期权价格上涨0.05元
```

## 例子
```
假设某投资者持有以下投资组合：
买入1份看涨期权(Delta=0.6, Gamma=0.03, Theta=-0.05, Vega=0.12, Rho=0.08)
卖出2份看跌期权(Delta=-0.4, Gamma=0.02, Theta=0.03, Vega=0.10, Rho=-0.05)

组合希腊值计算：
Delta = 1×0.6 + (-2)×(-0.4) = 0.6 + 0.8 = 1.4
Gamma = 1×0.03 + (-2)×0.02 = 0.03 - 0.04 = -0.01
Theta = 1×(-0.05) + (-2)×0.03 = -0.05 - 0.06 = -0.11
Vega = 1×0.12 + (-2)×0.10 = 0.12 - 0.20 = -0.08
Rho = 1×0.08 + (-2)×(-0.05) = 0.08 + 0.10 = 0.18

解释：
正Delta(1.4)：标的资产价格上涨有利
负Gamma(-0.01)：标的资产大幅波动不利
负Theta(-0.11)：时间流逝不利
负Vega(-0.08)：波动率上升不利
正Rho(0.18)：利率上升有利
这个组合适合预期标的资产价格温和上涨、波动率下降且利率上升的环境。
```


## 买入鹰式看涨期权（适用横盘震荡, 买卖卖买call）
买入鹰式看涨期权由四个期权交易构成
假设四个执行价满足：K₁ < K₂ < K₃ < K₄，且间距相等（通常K₂-K₁ = K₄-K₃）
```
操作	执行价	权利金	数量
买入Call	K₁	C₁	+1
卖出Call	K₂	C₂	-1
卖出Call	K₃	C₃	-1
买入Call	K₄	C₄	+1


净权利金支出/收入
净支出/收入 = (C₁ + C₄) - (C₂ + C₃)
若为正数：净支出（debit spread）
若为负数：净收入（credit spread）

最大收益
最大收益 = (K₂ - K₁) - 净支出
适用条件：到期时标的股价 ∈ [K₂, K₃]

最大风险
最大风险 = 净支出
适用条件：到期时标的股价 ≤ K₁ 或 ≥ K₄
```
盈亏平衡点
下方盈亏平衡点 = K₁ + 净支出
```
推导过程
当标的股价s处于 k1<s<k2
只有最低执行价 K1 的看涨期权为价内（可执行）
其他期权无价值
组合收益计算：
执行k1call得到 s-k1
净收益 = (s-k1) - 净支出
令净收益为0
得到 s = k1 + 净支出
```
上方盈亏平衡点 = K₄ - 净支出
```
推导过程
当k3<s<k4
k1k2k3为价内
k4为价外，无价值
组合计算
执行k1得到 s-k1 这是正收益
被执行k2得到 -(s-k2) 这是亏损
被执行k3得到 -(s-k3) 这是亏损
净收益 = (s-k1) - (s-k2) - (s-k3) - 净支出
化简得到
净收益 = (k2+k3-k1-s)-净支出
k之间有等差性质
k2-k1 = d
所以净收益 = (k3+d - s) - 净支出 
k3+d = k4
所以令净收益为0即得到
s = k4 - 净支出
```

分段收益函数
标的股价(S)区间	            收益公式
S ≤ K₁	                    -净支出
K₁ < S < K₂	                (S - K₁) - 净支出
K₂ ≤ S ≤ K₃	                (K₂ - K₁) - 净支出 (最大收益)
K₃ < S < K₄	                (K₄ - S) - 净支出
S ≥ K₄	                    -净支出
K₁ + 净支出<s<k4 - 净支出     正收益


例子
执行价：K₁=45, K₂=50, K₃=55, K₄=60（美元）
权利金：C₁=6, C₂=3, C₃=2, C₄=1（美元）
净支出 = (6 + 1) - (3 + 2) = 2美元
最大收益 = (50 - 45) - 2 = 3美元
最大风险 = 2美元
盈亏平衡点：
下方：45 + 2 = 47美元
上方：60 - 2 = 58美元

到期股价(S)	收益计算	                    结果
40	        -净支出 = -2	                -2 不行权损失权利金
47	        (47-45)-2 = 0 (下方平衡点)	    0   解释：超过45但是未超过50，k1期权生效以45买入结束(最多100股)
48	        (48-45)-2 = 1               	+1  解释：超过45但是未超过50，k1期权生效以45买入结束(最多100股)
...
50	        (50-45)-2 = 3 (最大收益区间)	+3  解释：超过50，被行权需要履行，用45买入卖出(最大100股, k1和k2对冲)
...
54          (50-45)-2 = 3 (最大收益区间)	+3  解释：超过50，被行权需要履行，用45买入卖出(最大100股, k1和k2对冲)
56          3-(56-55) = 2 	                +2  解释：超过55，被行权需要履行，用56买入55卖出(最大100股)
58	        3-(58-55) = 0 (上方平衡点)	    0   解释: 超过55，被行权需要履行，用58买入55卖出(最大100股)
60          3-(60-55) = -2 	                -2   解释: 60买入55卖出（k3和k4对冲），最后相当于损失权利金
65	        -净支出 = -2	                -2 60买入55卖出（k3和k4对冲），最后相当于损失权利金

```

标的：SPY ETF（当前价$52）
```

```
以2美元的价格买入1张行权价为54美元的call；

净收入 = 600 - 400 - 200 + 100 = 100
上平衡 = 48 + 1 = 49
下平衡 = 56 - 1 = 55

到期股价(S)	  收益计算	                         结果
48              100                             1   赚取权利金
49              -(49-48)*100+100=0              0   解释: 超过48，被行权以48卖出

58              -2+(58-54)-(58-56)+1=1          1   解释: 54和56对冲，约掉标的计算(56-54-2+1)，最终会赚取权利金
```

## 买入铁鹰式（适用大涨大跌， 卖买call买卖put）
```
构建
● 卖出1份call1
● 买入1份call2
● 买入1份put1
● 卖出1份put2

Call1、call2、put1、put2的标的资产、数量、到期日均相同，不同之处有：
行权价：call1>call2>put1>put2，且call1-call2=put1-put2；
上平衡点 = 买put行权价+净期权金
下平衡点 = 买call行权价+净期权金
最大收益 = 两端行权价间距+净期权金
最大亏损 = 净期权金

例子
以2美元的价格卖出1张行权价为56美元的call；
以3美元的价格买入1张行权价为52美元的call；
以2美元的价格买入1张行权价为48美元的put；
以1美元的价格卖出1张行权价为44美元的put.

净期权金 = (1+2) - (2+3) = -2

到期标的     收益                           结果
58          (58-52)-(58-56)-2 = 2          解释：56，52期权生效，56卖出，52买入，对冲，简化为(56-52-2)
54          (54-52) - 2 = 0                解释：52期权生效，52买入
52          -2                             解释：无期权生效，损失权利金
...                                        认为大涨大跌区间
48          -2                             解释：无期权生效，损失权利金
47          (48-47)-2 = -1                 解释：48期权生效，48卖出
44          (48-44)-2 = 2                  解释：48期权生效，48卖出
42          (48-42)-(44-42)-2 = 2          解释：48,44期权生效，48卖出，44买入，对冲，简化为(48-44-2)
```

## 卖出铁鹰式（适用变化不大，波动减缓 买卖call卖买put
```
构建
● 买入1份call1
● 卖出1份call2
● 卖出1份put1
● 买入1份put2

Call1、call2、put1、put2的标的资产、数量、到期日均相同，不同之处有：
行权价：call1>call2>put1>put2，且call1-call2=put1-put2；
上盈亏平衡点：股价=卖put行权价-净期权金
下盈亏平衡点：股价=卖call行权价+净期权金
最大收益 = 净期权金
最大亏损 = 同类型期权（call或者put）之间的行权价间距-净期权金

例子
买入1张行权价为56美元，报价为2美元的call；
卖出1张行权价为52美元，报价为3美元的call；
卖出1张行权价为48美元，报价为2美元的put；
买入一张行权价为44美元，报价为1美元的put.

净期权金 = 2
上平衡 = 48-2 = 46
下平衡 = 52+2 = 54
最大亏损 = 4 = 2  = 2

到期标的     收益                           结果
58          (58-56)-(58-52)+2 = -2         解释：56，52期权生效，52卖出，56买入，对冲，简化为(52-56+2)
54          -(54-52) + 2 = 0                解释：52期权生效，52卖出
52          -2                             解释：无期权生效，赚取权利金
...                                        认为平稳区间
48          -2                             解释：无期权生效，赚取权利金
47          -(48-47)+2 = 1                 解释：48期权生效，48买入
44          -(48-44)+2 = -2                解释：48期权生效，48买入
42          (44-42)-(48-42)+2 = -2         解释：48,44期权生效，48卖出，44买入，对冲，简化为(44-48+2)
```

# 平仓
## 买入期权后平仓（如买入看涨/看跌后卖出平仓）
```
开仓时：你支付权利金，获得行权权利（但无义务）。
平仓时：你卖出相同的期权合约，权利转移给新买家。
结果：
你不再拥有该期权的任何权利。
新买入该期权的人（你的对手方）获得行权权利
例子:
A从C买入后想结束
设定价格卖出平仓
B正在买入状态
预期匹配到A的
完成交易
A获得权利金后离场
B获得对C的权利
```

## 卖出期权后平仓（如卖出看涨/看跌后买入平仓）
```
开仓时：你收取权利金，承担履约义务（如被行权时必须按约定价格买卖标的）。
平仓时：你买入相同的期权合约，义务被对冲掉。
结果：
你不再有履约义务。
新卖出该期权的人（你的对手方）承担义务。
例子:
A卖给C一份合约想结束
设定价格买入平仓
B正在卖出状态
预期匹配到A的
完成交易
B获得权利金并承担对C的义务
A付出权利金后离场
```
