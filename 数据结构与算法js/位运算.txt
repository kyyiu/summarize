异或 ^: 二进制同位置相同为0，不同为1，任何数和0异或为本身，本身异或为0
也能理解成无进位的相加

满足交换，结合律，（无进位的相加理解更容易，某个位置的最终结果取决于出现的次数
a^b = b^a
(a^b)^c =a^(b^c)

可以使用这个，交换两个数
a = x
b = y
a = a ^ b // a此时为 x^y
b = a ^ b // b此时为 x^y^y = x
a = a ^ b // a此时为 x^y^x = y
但是前提是a，和b在内存中是两个独立的空间，不然会把自己变成0

提取二进制中最右边的1 ： x & (~x + 1)
假设 x 二进制 : 1010111100
~x: 0101000011
~x+1: 0101000100
x & (~x+1) : 0000000100
